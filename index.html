Зачем этот модуль

От скучного показа к живой сцене

Стандартный ImagePopout в Foundry просто открывает картинку. Art Presence делает появление персонажа событием.

Когда ГМ хочет показать злодея или важного NPC, он не просто кидает изображение — он режиссирует сцену. Персонаж появляется из-за края экрана, из угла, нависает сверху или выглядывает снизу. Вместе с ним — виньетка, частицы, туман, мерцание, дождь, фоновый звук.

Все карточки хранятся в библиотеке с папками и поиском. Один клик — и персонаж уже на экране у всех игроков. Другой клик — исчезает.

«Показывать злодея через обычный показ — как объявлять о появлении дракона через текстовый чат.»

Модуль написан с нуля под ApplicationV2 / DialogV2 (нативный Foundry V13 API), не использует сторонних зависимостей и работает через встроенные сокеты Foundry для синхронизации между всеми подключёнными клиентами.

🎭

Позиции на экране

12 готовых позиций для любого нарратива — от торжественного центра до тревожного нависания сверху. Плюс полностью ручная позиция с drag-редактором.

🌑

Настроения и эффекты

15 настроений с комбинациями виньетки, частиц, тумана, дождя, мерцания, тряски, пульса и анимированного тинта. От «Нейтрального» до «Безумия».

📚

Библиотека карточек

Сохраняйте персонажей в карточки с полным набором настроек. Организуйте по папкам, ищите по имени, показывайте одним кликом.

✒

Заголовки и шрифты

Имя персонажа поверх арта. Семь позиций, 20+ шрифтов включая кириллические, настройка размера, цвета, обводки и тени.

🔊

Фоновая музыка

К каждой карточке привязывается звуковой файл — тема персонажа играет в момент появления и плавно затухает при скрытии.

🌐

Синхронизация

Всё происходит у всех одновременно через WebSockets. Настройка аудитории: все, только ГМ, или конкретный игрок по ID.

  • Синхронизация через нативные сокеты Foundry
  • Нет внешних зависимостей
  • Игроки могут показывать арт персонажа

Аудитория

Кому показывать присутствие

Каждая карточка имеет настройку аудитории — кто увидит появление персонажа.

Значение Кто видит Типичное применение
all Все подключённые пользователи (ГМ + игроки) Публичное появление персонажа, битва, объявление
gm Только Мастер Игры Личная заметка ГМа, скрытый NPC
<userId> Конкретный игрок по ID пользователя Личное видение, шёпот одному игроку, секретная встреча

Макросы и API

Полный JavaScript API

Автоматизируйте показ персонажей из макросов, триггеров и других модулей.

Метод Описание Примечание
game.artPresence.openLibrary() Открыть окно библиотеки Только ГМ
game.artPresence.quickShow(path, opts) Быстрый показ по пути к файлу с опциями позиции, настроения, звука и аудитории Только ГМ
game.artPresence.show(card) Показать по объекту карточки из библиотеки Только ГМ
game.artPresence.hideAll() Скрыть все активные присутствия у всех Только ГМ
game.artPresence.library.getAll() Получить массив всех карточек библиотеки Чтение
game.artPresence.openPositionEditor(path) Открыть редактор позиции, возвращает Promise с координатами Только ГМ
📄 macros.js — примеры
// ── Два персонажа одновременно ──────────────────────────
const [ally, enemy] = ["Эйлин", "Мордан Тёмный"]
  .map(name => game.artPresence.library.getAll().find(c => c.name === name));

if (ally)  game.artPresence.show({ ...ally,  position: "left-lean" });
if (enemy) game.artPresence.show({ ...enemy, position: "right-lean" });

// ── Быстрый показ злодея ────────────────────────────────
game.artPresence.quickShow("worlds/campaign/boss.png", {
  position: "loom-top",
  mood:     "void",
  title:    "Архидемон",
  sound:    "music/boss-theme.ogg",
  audience: "all",
});
Настройки модуля

Четыре глобальные настройки

Все настройки в меню «Настройки Foundry → Параметры модуля».

Игроки могут показывать арты
При включении в листе персонажа появляется кнопка «Показать арт» — игрок может самостоятельно вывести арт своего персонажа на экраны остальных
Добавить кнопку к стандартному ImagePopout (В разработке)
При открытии любого встроенного ImagePopout добавляет кнопку «Показать через Art Presence» — удобная интеграция без изменения привычного workflow
Кнопки в панели сцены
Добавляет иконки «Библиотека» и «Скрыть все» в тулбар слоя токенов для быстрого доступа без горячих клавиш
Громкость звука (0–1)
Регулировка громкости фоновых звуков, привязанных к карточкам. Независимо от основного звука Foundry
